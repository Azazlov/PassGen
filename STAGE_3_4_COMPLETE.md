### 5.2 Фильтрация в StorageController

```dart
void _applyFilters() {
  _passwords = _allPasswords.where((entry) {
    // Фильтр по категории
    if (_selectedCategoryId != null && entry.categoryId != _selectedCategoryId) {
      return false;
    }
    // Поиск по сервису
    if (_searchQuery.isNotEmpty && !entry.service.toLowerCase().contains(_searchQuery)) {
      return false;
    }
    return true;
  }).toList();
  _currentIndex = 0;
  notifyListeners();
}
```

### 5.3 Группировка логов по дате

```dart
Map<String, List<SecurityLog>> getLogsByDate() {
  final Map<String, List<SecurityLog>> grouped = {};
  
  for (final log in _logs) {
    final date = _formatDate(log.timestamp);
    if (!grouped.containsKey(date)) {
      grouped[date] = [];
    }
    grouped[date]!.add(log);
  }
  
  return grouped;
}
```
