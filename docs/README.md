# PassGen Documentation

Project documentation is stored in this directory so source code and supporting materials stay separated.

## Main Documents

- `DEVELOPER.md` - detailed developer documentation
- `DOCUMENTATION.md` - documentation index and structure overview
- `PROJECT_STATUS_REPORT.md` - current project status
- `IMPLEMENTATION_REPORT.md` - implementation summary
- `FUNCTIONAL_TESTING_PLAN.md` - testing plan

## Subdirectories

- `assistant/` - AI assistant context files
- `chronology/` - development timeline and summaries
- `diploma/` - diploma-related materials

Repository root should stay focused on code, build configuration, and the primary `README.MD`.
