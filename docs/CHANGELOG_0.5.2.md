## 🔧 Исправления

### 1. Исправлена проблема с аутентификацией на macOS

**Проблема:**
- На платформе macOS не работала установка и проверка PIN-кода
- Ошибка: `no such table: auth_data` — таблица отсутствовала в существующей базе данных
- На Windows проблема не проявлялась

**Причина:**
- База данных создавалась с версией схемы 1 (без таблицы `auth_data`)
- При обновлении приложения миграция не применялась автоматически
- Флаг миграции из SharedPreferences уже был установлен, поэтому миграция пропускалась

**Решение:**
- Добавлена проверка существования таблицы `auth_data` в `main.dart`
- При отсутствии таблицы она создаётся автоматически при запуске приложения
- Добавлено подробное логирование для отладки работы с базой данных

**Изменённые файлы:**
- `lib/main.dart` — проверка и создание таблицы `auth_data`
- `lib/data/datasources/auth_local_datasource.dart` — улучшенное логирование
- `lib/data/database/database_helper.dart` — логирование пути к базе данных

**Тестирование:**
- ✅ Проверена установка PIN на macOS
- ✅ Проверена проверка PIN на macOS
- ✅ Проверена работа на Windows (регрессия)

### 2. Исправлен формат .passgen

**Проблема:**
- Некорректное использование nonce для ChaCha20-Poly1305
- Один 32-байтовый nonce использовался для двух целей (PBKDF2 и ChaCha20)

**Решение:**
- Разделение nonce на два независимых:
  - PBKDF2_NONCE: 32 байта (для деривации ключа)
  - CHACHA_NONCE: 12 байт (для шифрования)

**Изменённые файлы:**
- `lib/data/formats/passgen_format.dart`

## 🆕 Новые функции

### 1. История паролей

**Описание:**
- Новая таблица `password_history` для отслеживания изменений паролей
- Сохранение предыдущих версий паролей при обновлении
- Возможность просмотра истории изменений

**Таблица базы данных:**
```sql
CREATE TABLE password_history (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  entry_id INTEGER NOT NULL REFERENCES password_entries(id) ON DELETE CASCADE,
  service TEXT NOT NULL,
  encrypted_password BLOB NOT NULL,
  nonce BLOB NOT NULL,
  config TEXT NOT NULL,
  login TEXT,
  created_at INTEGER NOT NULL,
  reason TEXT
);
```

**Изменённые файлы:**
- `lib/data/database/database_schema.dart`
- `lib/data/database/database_migrations.dart`
- `lib/domain/entities/password_history_entry.dart`
