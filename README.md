# PassGen — Генератор паролей и система шифрования
## 📊 О проекте
PassGen — это кроссплатформенное приложение, разработанное на фреймворке Flutter с использованием языка Dart. Проект демонстрирует возможности создания безопасного решения для генерации и хранения паролей с применением современных криптографических методов.

## 🛠️ Технологии и инструменты
Flutter — для разработки кроссплатформенного интерфейса

Dart — основной язык программирования

Криптографические библиотеки — для генерации стойких чисел

Файловая система — для работы с локальным хранилищем

Сторонние библиотеки — для расширенной функциональности

## 💡 Основная концепция
Проект реализует уникальную систему хранения паролей, где вместо самих паролей сохраняются зашифрованные конфигурации их генерации. Это обеспечивает дополнительный уровень безопасности и надёжности.

## 📁 Части приложения
Генератор паролей — создание паролей по заданным параметрам

Шифрование сообщений — модуль для безопасной передачи данных

Хранилище конфигураций — система хранения зашифрованных настроек генерации

## 🎯 Функциональность
Основные возможности
Генерация паролей различной сложности

Шифрование и дешифрование сообщений

Безопасное хранение конфигураций

Кроссплатформенная поддержка

### Требования для компиляции исходных файлов

1. **Необходимые компоненты:**
   * Flutter SDK и Dart SDK для компиляции проекта
   * Android Studio (для сборки .apk под Android)
   * Xcode (для сборки .dmg и .ipa под iOS и macOS)

2. **Подготовка окружения:**
   ```bash
   flutter upgrade  # Обновление Flutter SDK
   flutter pub get   # Установка зависимостей
   ```

3. **Сборка проекта:**
   ```bash
   flutter build <платформа>  # Сборка для конкретной платформы
   ```
   *Поддерживаемые платформы:*
   * windows
   * linux
   * android (apk)
   * web

4. **Запуск в режиме отладки:**
   ```bash
   flutter run -d <платформа>  # Запуск в режиме реального времени
   ```
   *Поддерживаемые платформы для отладки:*
   * windows
   * linux

[Официальная документация по установке](https://docs.flutter.dev/get-started)

### Установка проекта из скомпилированных образов

#### Windows
* Распакуйте архив в удобное место
* Создайте ярлык на исполняемый файл для быстрого доступа

#### Linux
* **Вариант 1:** Распакуйте архив с бинарными файлами
* **Вариант 2:** Установите пакет .rpm (поддержка .deb планируется в будущих версиях)

#### Android
* Установите .apk файл
* Включите опцию установки из неизвестных источников

### Структура проекта

Все основные файлы проекта находятся в папке `lib`, которая содержит следующие модули:

* **app**
  * Точка входа в приложение
  * Система маршрутизации

* **features**
  * Отдельные папки для каждой страницы приложения
  * Файлы UI и логики взаимодействия

* **modules**
  * Основные модули приложения
  * Реализация бизнес-логики

* **shared**
  * Общие компоненты (виджеты, диалоги, интерфейсы)
  * Повторно используемые элементы

### Планы развития проекта

1. **Расширение функционала шифрования:**
   * Добавление поддержки шифрования файлов
   * Работа с бинарными данными
   * Повышение универсальности приложения

2. **Кастомизация настроек:**
   * Возможность настройки алгоритмов генерации паролей

   * Разработка сервера для хранения пользовательских конфигураций

   * Реализация синхронизации между устройствами
