## 5. ИНСТРУКЦИИ ДЛЯ ИИ-АГЕНТОВ

### 5.1 Перед началом работы

```bash
# 1. Проверь текущий прогресс
cat project_context/current_progress/CURRENT_PROGRESS.md

# 2. Ознакомься с планом
cat project_context/planning/WORK_PLAN.md

# 3. Проверь ТЗ
cat project_context/planning/passgen.tz.md
```

### 5.2 При выполнении задачи

1. **Создай план задачи:**
   ```
   project_context/planning/TASK_PLAN_YYYY-MM-DD.md
   ```

2. **Выполни задачу:**
   ```dart
   // [Выполнение задачи]
   ```

3. **За логируй результат:**
   ```
   project_context/logs/LOG_YYYY-MM-DD_TASK.md
   ```

### 5.3 После завершения задачи

1. **Обнови прогресс:**
   ```
   project_context/current_progress/CURRENT_PROGRESS.md
   ```

2. **Создай отчёт об этапе:**
   ```
   project_context/stages/STAGE_N_COMPLETE.md
   ```

3. **Проведи ревью:**
   ```
   project_context/reviews/CODE_REVIEW_YYYY-MM-DD.md
   ```
