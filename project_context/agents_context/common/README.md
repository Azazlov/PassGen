## 🎯 Быстрый старт

### Я пользователь

1. Прочитайте [Руководство пользователя](user_guide.md) для начала работы
2. Проверьте [FAQ](faq.md) если возникли вопросы

### Я разработчик

1. Изучите [Техническую документацию](technical/architecture.md)
2. Проверьте [structure.md](../../structure.md) для описания модулей

### Я на защите диплома

1. Откройте [Презентационные материалы](presentation/slides.md)
2. Используйте текст выступления и диаграммы
3. Подготовьте ответы на вопросы из раздела FAQ
