## 2. ТИПЫ ЛОГОВ

**Расположение:** `project_context/logs/LOG_YYYY-MM-DD_TOPIC.md`

**Назначение:** Логирование важных операций

## 8. АВТОМАТИЗАЦИЯ

### 8.1 Git hooks

**pre-commit hook:**
```bash
#!/bin/bash
# Автоматическое создание лога коммита

DATE=$(date +%Y-%m-%d)
LOG_FILE="project_context/logs/LOG_${DATE}_COMMIT.md"

if [ ! -f "$LOG_FILE" ]; then
  cat > "$LOG_FILE" << EOF
# 📝 Лог коммита: $DATE

## Изменения
$(git diff --cached --name-only)

## Сообщение
$(git diff --cached --format=%B)
EOF
fi
```

---

### 8.2 Скрипт создания лога

```bash
#!/bin/bash
# create_log.sh

TYPE=$1
DATE=$(date +%Y-%m-%d)

case $TYPE in
  merge)
    FILE="LOG_${DATE}_MERGE.md"
    ;;
  deploy)
    FILE="LOG_${DATE}_DEPLOY.md"
    ;;
  meeting)
    FILE="MEETING_${DATE}.md"
    ;;
  *)
    FILE="LOG_${DATE}.md"
    ;;
esac

cat > "project_context/logs/$FILE" << EOF
# 📝 Лог: $TYPE

**Дата:** $DATE

## Хронология

### $(date +%H:%M)
- 

## Итоги

EOF

echo "Created: project_context/logs/$FILE"
```
