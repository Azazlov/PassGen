# 📋 Product Manager Tracker — документация проекта

Эта директория содержит **актуальные управляющие документы проекта PassGen**: ТЗ, прогресс, финальный отчёт и результаты ревью/аудитов.

## Структура

```
project_context/product-manager-tracker/
├── planning/
│   └── passgen.tz.md
├── progress/
│   └── CURRENT_PROGRESS.md
├── stages/
│   └── FINAL_REPORT.md
└── reviews/
    ├── CODE_REVIEW_REPORT.md
    └── DATA_SECURITY_AUDIT.md
```

## Быстрые ссылки

- **Техническое задание**: `planning/passgen.tz.md`
- **Текущий прогресс**: `progress/CURRENT_PROGRESS.md`
- **Финальный отчёт**: `stages/FINAL_REPORT.md`
- **Код-ревью**: `reviews/CODE_REVIEW_REPORT.md`
- **Аудит безопасности**: `reviews/DATA_SECURITY_AUDIT.md`

## Примечание о “старой” структуре

Ранее в проекте использовалась папка `agents_context/` и набор “роль‑ориентированных” директорий (`qa_engineer/`, `devops_engineer/`, `data_security_specialist/`).
Сейчас навигация и “истина” по проекту — через `project_context/AGENTS.md` и документы в этой директории.
