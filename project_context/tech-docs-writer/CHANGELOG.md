# Changelog

Все значительные изменения в проекте PassGen задокументированы в этом файле.

## [0.5.1] — 10 марта 2026

### 🔐 Безопасность

- **Увеличены итерации PBKDF2** с 10,000 до 100,000 для повышенной защиты от brute-force атак
- **Включено логирование в транзакцию** для согласованности данных

### 🛡️ Надёжность

- **Добавлена проверка дубликатов** при импорте (service + login)
- **Обработка ошибок с rollback** — откат изменений при ошибке импорта

### ⚙️ Настройки

- **Настраиваемое время очистки буфера** — опция для изменения времени автоочистки (30 сек, 60 сек, 2 мин, 5 мин)

### 📊 База данных

- **Добавлена версия схемы БД** — константа `schemaVersion` для упрощения миграций
- **Документирование изменений схемы** — история версий в `database_schema.dart`

### 📝 Изменения в коде

| Файл | Изменения |
|---|---|
| `auth_local_datasource.dart` | PBKDF2: 10K → 100K итераций |
| `storage_local_datasource.dart` | Проверка дубликатов, rollback |
| `copyable_password.dart` | Параметр `clipboardTimeoutSeconds` |
| `database_schema.dart` | Версия схемы, документация |

## [0.5.0] — 10 марта 2026

### ✨ Новое

#### Генератор паролей
- **Опция «Без повторяющихся символов»** — каждый символ используется только один раз при генерации пароля
- **Опция «Исключить похожие символы»** — исключает символы `l`, `1`, `I`, `O`, `0` для избежания путаницы

#### Безопасность
- **Логирование PWD_ACCESSED** — запись в журнал событий при просмотре пароля
- **Логирование SETTINGS_CHG** — запись в журнал событий при изменении настроек приложения
- **Автоочистка буфера обмена** — пароль автоматически удаляется из буфера через 60 секунд после копирования

#### DevOps
- **Bash скрипты сборки** — автоматизация сборки для всех платформ
- **CI/CD GitHub Actions** — автоматические сборки, тесты, развёртывание
- **Скрипты развёртывания** — `deploy_test.sh`, `deploy_prod.sh`

### 🔧 Улучшения

#### Документация
- Обновлены все документы до версии 0.5.0
- Добавлена информация о новых функциях в руководство пользователя
- Обновлены технические спецификации
- Актуализированы презентационные материалы
- Добавлена документация по сборке и развёртыванию

#### Код
- Увеличено покрытие тестами до ~82%
- Рефакторинг компонентов генератора паролей
- Оптимизация работы с буфером обмена

## Формат

Этот проект следует [Semantic Versioning](https://semver.org/lang/ru/) и использует синтаксис [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

### Типы изменений

- **Added** — для новых функций
- **Changed** — для изменений в существующем функционале
- **Deprecated** — для функций, которые будут удалены в будущем
- **Removed** — для удалённых функций
- **Fixed** — для исправлений багов
- **Security** — для изменений в системе безопасности
