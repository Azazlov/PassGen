- **Design System:** Material 3

## 1. ЭКРАН АУТЕНТИФИКАЦИИ (AuthScreen)

### Назначение
Первичная аутентификация пользователя по PIN-коду (4-8 цифр).

### Макет (Mobile)
```
┌─────────────────────────────────┐
│                                 │
│         [🔐 Логотип]            │
│         PassGen                 │
│                                 │
│      Введите PIN-код            │
│    [●●●●] (4-8 цифр)           │
│                                 │
│  [1] [2] [3]                    │
│  [4] [5] [6]  ← Клавиатура      │
│  [7] [8] [9]                    │
│  [❌] [0] [⌫]                    │
│                                 │
│  ⚠️ Ошибка: "Неверный PIN"     │
│     [████░░░░] 30 сек блокировка│
│                                 │
└─────────────────────────────────┘
```

### Компоненты
| Компонент | Виджет | Описание |
|---|---|---|
| Логотип | `Icon(Icons.lock, size: 80)` | Центр экрана |
| Заголовок | `Text('Введите PIN-код')` | headlineMedium |
| PIN индикатор | `Row` с 8 ячеек | Заполненные круги (●) |
| Клавиатура | `GridView.builder (3x4)` | 12 кнопок (0-9, ❌, ⌫) |
| Кнопка входа | Авто-вход после 4 цифр | Без отдельной кнопки |
| Ошибка | `SnackBar` / `Banner` | Красный цвет, 30 сек блокировка |

### Поведение
| Действие | Реакция UI |
|---|---|
| Ввод цифры | Добавление точки (●), вибрация |
| Ввод 4+ символов | Авто-вход |
| 5 неудачных попыток | Блокировка 30 сек, прогресс-бар |
| Успешный вход | Переход на главный экран |
| Таймаут 5 мин | Возврат на экран авторизации |

### Особенности безопасности
- ✅ Маскировка ввода (● вместо цифр)
- ✅ Защита от скриншотов (FLAG_SECURE на Android)
- ✅ Очистка поля после ошибки
- ✅ Физическая клавиатура поддерживается

- `lib/presentation/features/auth/auth_screen.dart` (510 строк)
- `lib/presentation/features/auth/auth_controller.dart`
- `lib/presentation/features/auth/pin_input_widget.dart`

## 2. ЭКРАН ГЕНЕРАТОРА (GeneratorScreen)

### Назначение
Генерация криптографически стойких паролей по настраиваемым параметрам.

### Макет (Mobile)
```
┌─────────────────────────┐
│ 🔙 Назад  │ Генератор  │
├─────────────────────────┤
│                         │
│ [Сгенерированный пароль]│
│ ••••••••••••••••       │
│ [👁] [📋 Копировать]    │
│                         │
│ ───── Стойкость ─────   │
│ [████████░░] Надёжный  │
│                         │
│ ▼ Настройки генерации   │
│                         │
│ Длина: [8────●───64] 16│
│                         │
│ [✓] Строчные (a-z)     │
│ [✓] Заглавные (A-Z)    │
│ [✓] Цифры (0-9)        │
│ [✓] Спецсимволы (!@#)  │
│ [ ] Без повторов       │
│ [ ] Исключить похожие  │
│                         │
│ ▼ Пресеты              │
│ [Стандартный] [Надёжный]│
│ [Максимальный] [Свой+] │
│                         │
│ [🔄 Сгенерировать]      │
│ [💾 Сохранить в хранилище]│
└─────────────────────────┘
```

### Компоненты
| Компонент | Виджет | Описание |
|---|---|---|
| Результат | `CopyablePassword` | Карточка с паролем, копирование |
| Индикатор стойкости | `Slider` + `Text` | 5 уровней (Very Weak → Very Strong) |
| Слайдер длины | `Slider` (8-64) | С текстовым полем |
| Переключатели | `AppSwitch` (4 шт) | a-z, A-Z, 0-9, !@# |
| Опции | `AppSwitch` (2 шт) | Без повторов, Исключить похожие |
| Пресеты | `Slider` (0-4) | 5 уровней стойкости |
| Кнопки | `AppButton` (2 шт) | Сгенерировать, Сохранить |
| CharacterSetDisplay | Виджет | Отображение используемых символов |

### Параметры генерации
| Параметр | Диапазон | UI элемент |
|---|---|---|
| Длина пароля | 8-64 | Slider + TextField |
| Строчные буквы | Вкл/Выкл | AppSwitch |
| Заглавные буквы | Вкл/Выкл | AppSwitch |
| Цифры | Вкл/Выкл | AppSwitch |
| Спецсимволы | Вкл/Выкл | AppSwitch |
| Без повторов | Вкл/Выкл | AppSwitch |
| Исключить похожие | Вкл/Выкл | AppSwitch |

### Пресеты (5 уровней)
| Уровень | Конфигурация |
|---|---|
| 0 | Очень слабый (8 символов, только буквы) |
| 1 | Слабый (10 символов, буквы+цифры) |
| 2 | Средний (12 символов, все категории) |
| 3 | Надёжный (16 символов, все категории) |
| 4 | Максимальный (32 символа, уникальность) |
