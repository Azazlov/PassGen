ХАРАКТЕРИСТИКИ (Десктоп 900-1199dp):
──────────────────────────────────────────────────────────────────────────────────────────
• Макет: Трёхпанельный (NavigationRail + Список + Детали)
• Левая панель (80dp): NavigationRail с иконками 28px
• Средняя панель (280dp): Список паролей с поиском и фильтром
• Правая панель (Expanded): Детали выбранного пароля (max-width 800dp, центрировано)
• Все панели видны одновременно
• Вертикальные разделители: 1px между панелями

ВЗАИМОДЕЙСТВИЯ:
──────────────────────────────────────────────────────────────────────────────────────────
• Tap на элемент списка → Выделение элемента (показ деталей)
• Long press на элемент → Контекстное меню (удаление)
• Tap на пустое место → Сброс выделения
• Tap на 👁 → Показать/скрыть пароль
• Tap на 📋 → Копировать в буфер (очистка через 60 сек)
• Tap на ✏️ → Режим редактирования (открывается диалог)
• Tap на 🗑️ → Подтверждение удаления

══════════════════════════════════════════════════════════════════════════════════════════

АДАПТИВНОСТЬ:
──────────────────────────────────────────────────────────────────────────────────────────
• Контент центрирован с max-width 800dp
• При ширине экрана > 1200dp → переход на wide layout (3 панели одновременно)
• При ширине экрана < 900dp → переход на tablet layout (2 панели)
• При ширине экрана < 600dp → переход на mobile layout (1 панель)
