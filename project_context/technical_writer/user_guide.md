## Введение

**PassGen** — это кроссплатформенное приложение для генерации, хранения и управления паролями с использованием современных криптографических методов.

### Ключевые возможности

- 🔐 **Безопасное хранение** — все пароли шифруются алгоритмом ChaCha20-Poly1305
- 🎲 **Генератор паролей** — создание надёжных паролей разной сложности
- 📦 **Импорт/экспорт** — резервное копирование в JSON и фирменном формате .passgen
- 🔒 **PIN-защита** — доступ к приложению защищён PIN-кодом
- 📝 **Журнал событий** — отслеживание всех действий безопасности
- ✨ **Новое в v0.5.0:** опции «Без повторов» и «Исключить похожие», автоочистка буфера

### Поддерживаемые платформы

- Windows
- Linux
- Android

## Начало работы

### Установка

#### Windows

1. Скачайте архив `pass_gen_v0.5.0.zip` с [страницы релизов](https://github.com/azazlov/passgen/releases)
2. Распакуйте архив в удобную папку (например, `C:\Programs\PassGen`)
3. Запустите `pass_gen.exe`
4. **Требования:** Windows 10+, 4GB RAM, 100MB свободного места

> ⚠️ **Примечание:** Если приложение не запускается, установите [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

#### Linux

1. Скачайте архив `passgen_v0.5.0.tar.gz`
2. Распакуйте: `tar -xzf passgen_v0.5.0.tar.gz`
3. Запустите: `./pass_gen`
4. **Требования:** Ubuntu 18.04+, 4GB RAM, 100MB свободного места
5. **Зависимости:** `sudo apt-get install libgtk-3-dev`

#### Android

1. Скачайте `passgen_v0.5.0.apk`
2. Разрешите установку из неизвестных источников (Настройки → Безопасность)
3. Установите APK
4. **Требования:** Android 8.0+, 2GB RAM

### Первый запуск

При первом запуске вам будет предложено установить **PIN-код**:

1. Введите PIN-код от 4 до 8 цифр
2. Подтвердите PIN-код повторным вводом
3. После успешной установки вы попадёте в главное меню

> ⚠️ **Важно:** Запомните ваш PIN-код! При его утере доступ к приложению будет невозможен.

## Генератор паролей

### Быстрая генерация

1. Перейдите на вкладку **Генератор**
2. Выберите категорию (например, "Социальные сети")
3. Нажмите кнопку **"Сгенерировать пароль"**
4. Скопируйте результат или сохраните в хранилище

### Настройка сложности

Приложение предлагает 5 пресетов сложности:

| Уровень | Описание | Пример |
|---------|----------|--------|
| 1️⃣ **Стандартный** | Базовая защита | `aB3xYz` |
| 2️⃣ **Средний** | Рекомендуется | `aB3xYz9K` |
| 3️⃣ **Высокий** | Для важных аккаунтов | `aB3xYz9K@m` |
| 4️⃣ **Очень высокий** | Максимальная защита | `aB3xYz9K@mNp` |
| 5️⃣ **Максимальный** | Критические данные | `aB3xYz9K@mNp$7Qw` |

### Параметры генерации

- **Длина пароля:** от 8 до 64 символов
- **Набор символов:**
  - Строчные буквы (a-z)
  - Заглавные буквы (A-Z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*)
- **Дополнительные опции (v0.5.0):**
  - ✅ **Без повторяющихся символов** — каждый символ используется только один раз
  - ✅ **Исключить похожие символы** — исключает символы l, 1, I, O, 0 для избежания путаницы

### Оценка надёжности

После генерации пароль автоматически оценивается по шкале:

- 🔴 **Слабый** — легко подобрать
- 🟠 **Средний** — приемлемо
- 🟡 **Хороший** — надёжно
- 🟢 **Очень надёжный** — отлично
- 🟣 **Максимальный** — невозможно подобрать

### Сохранение пароля

1. Сгенерируйте пароль
2. Введите название сервиса (например, "Gmail")
3. При необходимости укажите логин
4. Выберите категорию
5. Нажмите **"Сохранить"**

## Хранилище паролей

### Просмотр паролей

1. Перейдите на вкладку **Хранилище**
2. Используйте навигацию для просмотра записей
3. Нажмите на запись для просмотра деталей

### Поиск и фильтрация

**Поиск по названию:**
1. Введите название сервиса в поле поиска
2. Результаты фильтруются автоматически

**Фильтр по категориям:**
1. Нажмите на иконку категории
2. Выберите нужную категорию
3. Отобразятся только пароли этой категории

### Категории

В приложении доступны 7 системных категорий:

- 🌐 **Интернет** — сайты и онлайн-сервисы
- 📱 **Социальные сети** — VK, Telegram, WhatsApp
- 🎮 **Развлечения** — игры, стриминговые сервисы
- 💼 **Работа** — рабочие аккаунты
- 💰 **Финансы** — банки, платёжные системы
- 🛒 **Покупки** — интернет-магазины
- 📧 **Почта** — email-сервисы

### Копирование пароля

- Нажмите на пароль для копирования в буфер обмена
- Появится уведомление об успешном копировании
- **Автоочистка буфера (v0.5.0):** пароль автоматически удаляется из буфера через 60 секунд

### Удаление пароля

1. Выберите запись
2. Нажмите кнопку **"Удалить"**
3. Подтвердите действие

> ⚠️ **Внимание:** Удаление необратимо!

## Шифратор сообщений

### Шифрование текста

1. Перейдите на вкладку **Шифратор**
2. Выберите режим **"Зашифровать"**
3. Введите текст сообщения
4. Введите пароль для шифрования
5. Нажмите **"Зашифровать"**
6. Скопируйте результат

### Дешифрование текста

1. Перейдите на вкладку **Шифратор**
2. Выберите режим **"Дешифровать"**
3. Вставьте зашифрованный текст
4. Введите тот же пароль
5. Нажмите **"Дешифровать"**
6. Скопируйте расшифрованное сообщение

> 🔐 **Важно:** Используйте надёжные пароли для шифрования!

## Настройки

### Управление PIN-кодом

**Смена PIN:**
1. Перейдите в **Настройки**
2. Нажмите **"Сменить PIN"**
3. Введите текущий PIN
4. Введите новый PIN (4-8 цифр)
5. Подтвердите новый PIN

**Удаление PIN:**
1. Перейдите в **Настройки**
2. Нажмите **"Удалить PIN"**
3. Введите текущий PIN для подтверждения

> ⚠️ **Предупреждение:** После удаления PIN доступ к приложению будет возможен без аутентификации!

### Журнал событий

Для просмотра журнала безопасности:

1. Перейдите на вкладку **Журнал**
2. Просмотрите список событий

Типы событий:
- ✅ **AUTH_SUCCESS** — успешная аутентификация
- ❌ **AUTH_FAIL** — неудачная попытка входа
- 🔓 **LOGOUT** — выход из приложения
- 📝 **PWD_CREATED** — создание пароля
- 🗑️ **PWD_DELETED** — удаление пароля
- 👁️ **PWD_ACCESSED** — просмотр пароля (v0.5.0)
- 📤 **EXPORT** — экспорт данных
- 📥 **IMPORT** — импорт данных
- ⚙️ **SETTINGS_CHG** — изменение настроек (v0.5.0)

## Частые проблемы

### ❌ Неверный PIN-код

**Проблема:** Приложение сообщает о неверном PIN-коде.

**Решение:**
- Проверьте раскладку клавиатуры
- Убедитесь, что не нажат Caps Lock
- После 5 неудачных попыток доступ блокируется на 30 секунд

### ❌ Не могу скопировать пароль

**Проблема:** Кнопка копирования не работает.

**Решение:**
- Проверьте разрешения приложения
- Попробуйте скопировать пароль вручную
- Перезапустите приложение

### ❌ Данные не сохраняются

**Проблема:** Пароли не сохраняются в хранилище.

**Решение:**
- Проверьте свободное место на устройстве
- Убедитесь, что приложение имеет доступ к хранилищу
- Перезапустите приложение

### ❌ Ошибка при импорте

**Проблема:** Не удаётся импортировать файл.

**Решение:**
- Проверьте формат файла (JSON или .passgen)
- Убедитесь, что файл не повреждён
- Для .passgen проверьте мастер-пароль

### ❌ Приложение закрывается при неактивности

**Проблема:** Приложение автоматически блокируется.

**Решение:**
- Это нормальное поведение (защита от несанкционированного доступа)
- Таймер неактивности: 5 минут
- Введите PIN-код для разблокировки
