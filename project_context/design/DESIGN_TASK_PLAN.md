## 3. ОТКРЫТЫЕ ЗАДАЧИ UI/UX (ПО ПРИОРИТЕТАМ)

### 🔴 Критические (обязательно по ТЗ)

#### Задача 1: Адаптивная навигация
**Приоритет:** 🔴 Критический
**Оценка:** 8 часов
**ТЗ раздел:** 3.2, 3.4

**Описание:**
Реализовать адаптивную навигацию для мобильных/планшетов/десктопов.

**Требования:**
- Мобильный (<600dp): BottomNavigationBar
- Планшет (600-900dp): NavigationRail
- Десктоп (>900dp): NavigationRail + Sidebar

**Что сделать:**
1. Создать макеты для 3 типов устройств
2. Экспортировать в `final/navigation_[mobile/tablet/desktop].png`
3. Обновить `components.json` с адаптивными спецификациями
4. Передать разработчикам (обновить `for_development/`)

**Результат:**

   ```
   grep -A 20 "Раздел [X]" project_context/planning/passgen.tz.md
   ```

2. **Проверь текущие гайдлайны:**
   ```
   cat project_context/design/guidelines/guidelines.md
   ```

3. **Проверь for_development:**
   ```
   ls project_context/design/for_development/
   ```

### 7.2 Во время выполнения
1. **Создай прототип:**
   ```
   project_context/design/prototypes/[task]_v1.fig
   ```

2. **Экспортируй макет:**
   ```
   project_context/design/final/[task].png
   ```

3. **Обнови JSON:**
   ```
   project_context/design/for_development/[component].json
   ```

### 7.3 После завершения
1. **Обнови changelog:**
   ```markdown
   ## [1.1] - 2026-03-08

   ### Добавлено
   - [Новый компонент]

   ### Изменено
   - [Изменённый компонент]
   ```

2. **Обнови guidelines.md** (если изменились компоненты)

3. **Закоммить изменения:**
   ```bash
   git add project_context/design/
   git commit -m "design: [описание изменений]"
   ```
