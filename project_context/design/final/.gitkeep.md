# Final Exported Designs Placeholder

This folder contains exported final designs in PNG and PDF formats.

## File Naming Convention

`[screen_name]_[theme]_[size].[format]`

Examples:
- `auth_screen_light.png`
- `auth_screen_dark.png`
- `generator_screen_light.png`
- `storage_screen_light.pdf`

## Export Settings

### PNG
- Resolution: 2x (for Retina displays)
- Background: Transparent (where applicable)
- Color Profile: sRGB

### PDF
- Vector format (scalable)
- All fonts embedded
- CMYK color profile for print

## Screens to Export

For each screen, export:
1. Light theme variant
2. Dark theme variant
3. Mobile size (375x812 or similar)
4. Tablet size (768x1024 or similar)
5. Desktop size (1920x1080 or similar)
