### 2.3 Реализована ротация ключей при смене PIN

**Файл:** `lib/data/datasources/auth_local_datasource.dart`

**Процесс ротации:**

```dart
Future<bool> changePin(String oldPin, String newPin, {Database? database}) async {
  // 1. Проверка старого PIN
  final verified = await verifyPin(oldPin);
  
  // 2. Ротация ключей шифрования
  if (db != null) {
    await _rotateEncryptionKeys(oldPin, newPin, db);
  }
  
  // 3. Установка нового PIN
  return await setupPin(newPin);
}
```

**Алгоритм `_rotateEncryptionKeys`:**
1. Получить соль старого PIN
2. Derive старого ключа шифрования
3. Получить все зашифрованные пароли из БД
4. Для каждого пароля:
   - Расшифровать старым ключом
   - Derive нового ключа шифрования
   - Зашифровать новым ключом
   - Обновить запись в БД
   - Затереть расшифрованный пароль
5. Затереть старые ключи

**Результат:**
- ✅ Полная ротация ключей при смене PIN
- ✅ Затирание старых ключей
- ✅ Безопасное промежуточное хранение
