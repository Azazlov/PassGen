### 2.2 Создана система версионирования шифрования

**Файл:** `lib/core/utils/encryption_versioning.dart`

**Реализованные классы:**

#### EncryptionVersion
| Константа | Значение | Описание |
|---|---|---|
| `currentVersion` | 1 | Текущая версия |
| `minSupportedVersion` | 1 | Минимальная поддерживаемая |
| `v1` | 1 | ChaCha20-Poly1305 + PBKDF2 (10000) |
| `v2` | 2 | ChaCha20-Poly1305 + PBKDF2 (20000) |
| `v3` | 3 | ChaCha20-Poly1305 + Argon2id |

#### EncryptionParams
| Параметр | v1 | v2 | v3 |
|---|---|---|---|
| Алгоритм | ChaCha20-Poly1305 | ChaCha20-Poly1305 | ChaCha20-Poly1305 |
| KDF | PBKDF2-HMAC-SHA256 | PBKDF2-HMAC-SHA256 | Argon2id |
| Итерации | 10,000 | 20,000 | 3 |
| Длина ключа | 256 бит | 256 бит | 256 бит |

#### EncryptionMetadata
**Назначение:** Метаданные шифрования для файла/записи

**Поля:**
- `version` — версия алгоритма
- `timestamp` — время создания
- `keyId` — идентификатор ключа (для ротации)
- `extra` — дополнительные данные

**Методы:**
- `toJson()` — сериализация
- `fromJson()` — десериализация
- `isCompatible` — проверка совместимости
- `params` — получение параметров версии
